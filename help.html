It's a short introduction in the API

Available URLs:

1. https://audio-editor-api.herokuapp.com

Methods: GET

Description: base url, return information about this API.

2. https://audio-editor-api.herokuapp.com/trim/start_time=start_time&end_time=end_time

Parameters:

  • start_time(int): Start time of the clip (seconds).
  • end_time(int): End time of the clip (seconds).

Methods: POST

Description: Excerpt a clip from an audio file, given the start timestamp and end timestamp of the clip within the file, expressed in seconds.

3. https://audio-editor-api.herokuapp.com/treble/gain=gain

Parameters:

  • gain(int): The gain at the Nyquist frequency. For a large cut use -20, for a large boost use 20.

Methods: POST

Description: Boost or cut the treble (lower) frequencies of the audio using a two-pole shelving filter with a response similar to that of a standard hi-fi’s tone-controls. This is also known as shelving equalisation.

4. https://audio-editor-api.herokuapp.com/reverse

Methods: POST

Description: Reverse the audio completely.

5. https://audio-editor-api.herokuapp.com/concatenate

Methods: POST

Description: Concatenate several files into one.

6. https://audio-editor-api.herokuapp.com/mix

Methods: POST

Description: Mix several files into one.

7. https://audio-editor-api.herokuapp.com/fade/fade_start=fade_start&fade_end=fade_end

Parameters:

  • fade_start(float): Length of fade-in (seconds). If fade_start = 0.0, no fade in is applied.
  • fade_end(float): Length of fade-out (seconds). If fade_end = 0.0, no fade in is applied.

Methods: POST

Description: Add a fade in and/or fade out to an audio file.

8. https://audio-editor-api.herokuapp.com/flanger/effect=effect

Parameters:

  • effect(string): one of theme - low, medium, high.

Methods: POST

Description: Apply a flanging effect to the audio.

9.1 https://audio-editor-api.herokuapp.com/tremolo

9.2 https://audio-editor-api.herokuapp.com/tremolo/speed=speed&depth=depth

Parameters:

  • speed(int, default=6): Sweeps per second (in Hz) between 1 and 10.
  • depth(int, default=50): Added swept delay (in milliseconds) between 0 and 10.

Methods: POST

Description: Apply a flanging effect to the audio.

10. https://audio-editor-api.herokuapp.com/volume/new_volume=new_volume

Parameters:

  • new_volume(string): value in decibels, max=15, min=-50.

Methods: POST

Description: Apply an amplification or an attenuation to the audio signal.

11. https://audio-editor-api.herokuapp.com/info

Methods: POST

Description: Get information about file (channels, sample rate, encoding, duration, size).

12. https://audio-editor-api.herokuapp.com/chorus/number_of_voices=number_of_voices

Parameters:

  • number_of_voices(int): The number of voices in the chorus effect, max=5.

Methods: POST

Description: Add a chorus effect to the audio. This can make a single vocal sound like a chorus, but can also be applied to instrumentation.

13. https://audio-editor-api.herokuapp.com/echo

Parameters (send as data):

  • gain_in(float): Input volume, between 0 and 1.
  • gain_out(float): Output volume, between 0 and 1.
  • n_echos(int): Number of reflections.

Methods: POST

Description: Add echoing to the audio.

14. https://audio-editor-api.herokuapp.com/bass

Parameters (send as data):

  • gain_db(float): The gain at 0 Hz. For a large cut use -20, for a large boost use 20.
  • frequency(float): The filter’s cutoff frequency in Hz.
  • slope(float): The steepness of the filter’s shelf transition. For a gentle slope use 0.3, and use 1.0 for a steep slope.

Methods: POST

Description: Boost or cut the bass (lower) frequencies of the audio using a two-pole shelving filter with a response similar to that of a standard hi-fi’s tone-controls.

15. https://audio-editor-api.herokuapp.com/speed/new_speed=new_speed

Parameters:

  • new_speed(float): The ratio of the new speed to the old speed. For ex. 1.1 speeds up the audio by 10%; 0.9 slows it down by 10%.

Methods: POST

Description: Adjust the audio speed.

16. https://audio-editor-api.herokuapp.com/repeat/n=n

Parameters:

  • n(int): The number of times to repeat the audio.

Methods: POST

Description: Repeat the entire audio count times.

17. https://audio-editor-api.herokuapp.com/convert/new_format=new_format

Parameters:

  • new_format(string): Available formats - ['.aif', '.aifc', '.aiff', '.au', '.avr', '.flac', '.ogg', '.snd', '.wav'].

Methods: POST

Description: Convert file into selected format.

P.S: Send file in request with key 'file', max size files = 20Mb
